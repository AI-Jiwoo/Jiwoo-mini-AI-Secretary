# Jiwoo
> ## 창업 지원 AI서비스
> 타겟 시장
> - 초기 창업자, 스타트업, 중소기업, 벤처 투자자 등<br>
> 
> 타겟 고객 
> - 창업을 준비 중이거나 이미 사업을 시작했지만 추가적인 데이터 분석과 시장 조사를 통해 사업을 성장시키고자 하는 창업자
> 
> 서비스 가치
> - 사업에 투자되는 시간과 비용 절감
> - 사업 방향 결정에 필요한 신뢰도 있는 데이터 제공(수요, 공급 동향 등)
> - 사업 분석을 통해 공략할 수 있는 틈새 사업 도출
> - 사업 아이템의 객관화(상업성의 가치 판단)
